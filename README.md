# Teams-clone
This project is deployed at https://teamsclonebyshriyanshi.herokuapp.com/

# Features
* Input Name
* Copy Invite Link
* Audio Mute/Unmute
* Video On/Off
* Share Screen
* Chat Feature
* Group Call

# Agile Implementation
* Learn
* Chose the tools and technologies and designed a basic prototype
* Add more features + Test and fix bugs

# Technologies used
* Node express server
* React frontend
* WebRTC
* Socket.io
* Simple-peer
* Bootstrap

# Setup
`git clone git@github.com:shriyanshibhadada/Project.git`
### Server setup
`npm i`\
`nodemon`

### client setup
`cd teams-clone`\
`npm i`\
`npm start`
